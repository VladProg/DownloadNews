Компанія Microsoft планує впровадити свого персонального асистента на основі штучного інтелекту Copilot для операційної системи Windows 10.

Про це повідомляє Engadget, передає Укрінформ.
Як зазначається, кнопка і бічна панель Copilot з Windows 11 скоро з'явиться на панелі завдань Windows 10. Під час натискання на кнопку відкриватиметься той самий інтерфейс бічної панелі Copilot, що і у Windows 11.
Асистент зі штучним інтелектом для Windows 11 був запущений в бета-версії у серпні, а офіційно – з вересня. Windows 10 Copilot включатиме плагіни, які працюватимуть в обох операційних системах.
Як повідомляє видання, однією з головних причин, чому Microsoft впроваджує можливості штучного інтелекту у Windows 10, є частка ринку: порівняно з 40 млн користувачів Windows 11 попередню версію Windows використовують близько 1 млрд людей.
У звіті Windows також зазначається, що Microsoft переглядає свою стратегію швидкого переходу від Windows 10. Повідомляється, що компанія планує додати в стару ОС "деякі нові функції і служби", які раніше були ексклюзивними для Windows 11. Підтримка Windows 10 закінчується в жовтні 2025 року, але в компанії тривають внутрішні дискусії про можливість її продовження.
Як повідомляв Укрінформ, технологія, яка є основою системи штучного інтелекту ChatGPT, буде додана до робочого програмного забезпечення Microsoft 365.
