Microsoft на нетривалий час забороняла своїм працівникам користуватися ChatGPT, зокрема закривши доступ до нього на корпоративних пристроях.

Про це повідомляє CNBC, передає Укрінформ.
"Через проблеми з безпекою та даними низка інструментів штучного інтелекту більше недоступні для використання співробітниками", - йшлося у повідомленні на внутрішньому сайті Microsoft.
Компанія заявила, що хоч Microsoft інвестував у OpenAI і ChatGPT має вбудовані засоби захисту для запобігання неналежному використанню, все ж "вебсайт є стороннім зовнішнім сервісом", тож працівникам компанії рекомендують "бути обережними під час його використання через ризики для конфіденційності та безпеки". Рекомендація також стосується і інших зовнішніх сервісів ШІ, таких як Midjourney або Replika.
Згодом у Microsoft заявили, що тимчасове блокування ChatGPT було помилкою, яка виникла в результаті тестування систем для великих мовних моделей.
"Ми тестували системи контролю кінцевих точок для LLM і ненавмисно ввімкнули їх для всіх співробітників. Ми відновили роботу незабаром після того, як виявили помилку. Закликаємо співробітників і клієнтів використовувати такі сервіси, як Bing Chat Enterprise і ChatGPT Enterprise, які мають більш високий рівень захисту конфіденційності та безпеки", - сказав представник компанії.
Оновлення Microsoft рекомендує людям використовувати власний інструмент Bing Chat на основі моделі штучного інтелекту OpenAI. Цього року Microsoft повідомила про випуск оновлень ОС Windows і додатків Office, які використовують переваги сервісів OpenAI, які, в свою чергу, працюють на хмарній інфраструктурі Microsoft Azure.
Як повідомляв Укрінформ, Microsoft планує впровадити свого персонального асистента на основі штучного інтелекту Copilot для операційної системи Windows 10.
