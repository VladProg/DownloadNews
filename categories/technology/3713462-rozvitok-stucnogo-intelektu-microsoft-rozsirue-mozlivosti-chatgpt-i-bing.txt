Корпорація Microsoft почала надавати користувачам безліч оновлень ШІ, включаючи ChatGPT, пошукову систему Bing, а також хмарні сервіси.

Як передає Укрінформ, про це повідомляє Reuters.
Зокрема, серед ключових змін впроваджується розгортання реальних результатів пошуку з Bing у ChatGPT, відповіді якого спочатку були обмежені інформацією станом на 2021 рік.
"Тепер ChatGPT може отримувати вебрезультати з Bing для передплатників і незабаром зробить це для безкоштовних користувачів", - повідомила компанія.
Microsoft також розширює так звані плагіни для Bing, використовуючи стандарт OpenAI, що дозволяє компаніям легше взаємодіяти зі споживачами в пошуковій системі.
"Це глибока зміна в тому, як люди будуть використовувати Інтернет", - сказав директор з маркетингу для споживачів Microsoft Юсуф Мехді.
Оновлення Bing є частиною зусиль Microsoft, спрямованих на те, щоб захопити більшу частину світового ринку пошукової реклами, який оцінюється у 286 мільярдів доларів.
Нові функції хмарного сервісу дозволяють компаніям створювати плагіни, що підключаються до асистента зі штучним інтелектом для підприємств Microsoft 365 Copilot.
За словами представників Microsoft, плагін може дозволити співробітнику простою мовою попросити ШІ забронювати поїздку або пояснити юридичні питання, пов'язані з контрактами з постачальниками.
Компанія також заявила, що з червня планується розробка ШІ-помічника, доступного як тестова версія для деяких користувачів операційної системи Windows.
Як повідомляв Укрінформ, Білий дім дізнаватиметься у працівників, як їхні роботодавці використовують штучний інтелект (ШІ).
