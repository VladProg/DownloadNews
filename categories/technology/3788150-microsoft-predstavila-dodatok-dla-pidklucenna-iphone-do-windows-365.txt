Корпорація Microsoft випустила програму Windows App для iOS, iPadOS, macOS, Windows і браузерів. Вона призначена для віддаленого підключення до продуктів виробника софту.

Про це повідомляє The Verge, передає Укрінформ.
Додаток створили на основі Windows 365 і перетворили його на центральний вузол підключення з віддалених ПК, віртуального робочого столу Azure, Windows 365, Microsoft Dev Box і служб віддаленого робочого столу Microsoft.
В оновленому додатку доступні налаштування роздільної здатності і масштабування дисплея, а також підключення периферійних пристроїв, таких як вебкамери, накопичувачі і принтери.
Попередня версія програми для Windows наразі недоступна для Android.
Користуватися додатком можуть лише власники бізнес-акаунтів Microsoft, але є ймовірність того, що він стане доступним і для звичайних користувачів.
Microsoft має подібні програми для віддаленого підключення до комп'ютерів, однак окремий додаток є незвичним брендингом і може свідчити про плани Microsoft щодо перенесення Windows у хмару.
Загалом у Windows 11 уже з'явилася низка вебфункцій, зокрема головний пошуковий інтерфейс, який оновлюється з інтернету, система віджетів та інтеграція з Copilot.
Як повідомляв Укрінформ, компанія Microsoft планує впровадити свого персонального асистента на основі штучного інтелекту Copilot для операційної системи Windows 10.
