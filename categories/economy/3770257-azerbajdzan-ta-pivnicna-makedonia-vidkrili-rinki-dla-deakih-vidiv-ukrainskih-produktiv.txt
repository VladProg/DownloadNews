Державна служба України з питань безпечності харчових продуктів та захисту споживачів досягла домовленостей стосовно експорту нових видів продукції українського виробництва в Азербайджанську Республіку та в Республіку Північна Македонія.

Як передає Укрінформ, про це повідомляє пресслужба Держпродспоживслужби.
Зазначається, що з Агентством харчової безпеки Азербайджанської Республіки погоджено форму ветеринарного сертифіката на композитні продукти з вмістом м'яса птиці та молочних продуктів, доведених до кулінарної готовності, призначених для споживання людиною, що експортуються до Азербайджанської Республіки.
З Агентством з продовольства та ветеринарії Міністерства сільського, лісового та водного господарства Республіки Північна Македонія погоджено форму сертифіката здоров’я для перероблених кормів для домашніх тварин, крім консервованих кормів для домашніх тварин, призначених для відправлення до або для транзиту через Республіку Північна Македонія.
Видача сертифіката здоров’я для перероблених кормів для домашніх тварин, крім консервованих кормів для домашніх тварин, призначених для відправлення до або для транзиту через Республіку Північна Македонія може здійснюватися лише за умови наявності виробника у переліку потужностей, акредитованих для експорту м’ясних напівфабрикатів до ЄС.
Як повідомляв Укрінформ, Державна служба України з питань безпечності харчових продуктів та захисту споживачів у 2023 році веде роботу з відкриття понад 27 нових міжнародних ринків для українських товарів.
